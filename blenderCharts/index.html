Animated Charts in Blender

For a recent project, I needed to create high-quality animated charts for video. The charts themselves weren't that complex – I mostly needed to create column and line charts – but it was important that they were customizable, repeatable, and looked good.

I was a little disappointed to discover that the most common way to do this was by hand. There's a whole subgenre of tutorials on Youtube explaining how to manually create charts using After Effects, but not much in the way of how to do this programmatically. There was nothing like d3.js for video.

So I turned to Blender, the open-source Swiss Army Knife for modeling, animating, and rendering to create my own tools. Blender has its fair share of issues, but one of the great things about it is that literally anything you want to do can be scripted in Python.

I created a quick Python script to create basic charts using Blender's built-in primitives (primary Text, Cubes, Lights, and Cameras). I needed an example data set, so I grabbed some data from Fangraphs, a hyper-nerdy baseball statistics site.
