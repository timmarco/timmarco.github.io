One Way To Visualize Functions With a Long Headline Title

The most common way to think about functions is to use a Cartesian Plot.

Cartesian plots are two-dimensional graphs that show the input value horizontally (on the x-axis), and the output value vertically (on the y-axis).

Here's a Cartesian Plot of the function \[f(x) = 1 + \frac{1}{x} \].

Another Way to Visualize

But that isn't the only way to visualize functions. You can also think of functions as transforms.

Here, you can think of separate number lines for the input and the output. Instead of a single line mapping between the two, you can draw many lines, one from every input to every output.

So for example, if the input were 2.4, the output would be [value].

When you visualize functions as transforms, the pattern that matters is how the lines map.
